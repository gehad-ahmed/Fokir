# 🎨 Fokir — Personal Portfolio Landing Page

> 🔗 **Live Demo:** [https://gehad-ahmed.github.io/Fokir/](https://gehad-ahmed.github.io/Fokir/)

A responsive personal-portfolio landing page with a modern layout for showcasing skills, services and work, built with HTML and CSS.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

## 📖 Overview

Fokir is a sleek single-page portfolio site featuring a hero introduction, about, services, portfolio and contact sections — designed to be fully responsive and easily customizable.

## ✨ Features

- 👋 Hero / introduction section
- 🧑‍💻 About and skills
- 🛠️ Services and portfolio showcase
- ✉️ Contact section
- 📱 Fully responsive layout

## 🛠️ Tech Stack

- HTML5 & CSS3

## 🚀 Getting Started

```bash
git clone https://github.com/gehad-ahmed/Fokir.git
cd Fokir
```

Then open `index.html` in your browser (or use the VS Code **Live Server** extension).
